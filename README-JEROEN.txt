Dag Jeroen 

Bij deze mijn website voor het bijzonder vraagstuk Building Informatics 

Enkele zaken: 

Ik heb geprobeerd met een pod te werken maar kreeg het niet werkende, misschien kunnen we hier volgend 
semester nog eens naar kijken? Voor nu heb ik het dus zonder gedaan, mijn afbeeldingen bevinden zich in 
een Assets folder.

Bij het verkleinen van het window verandert de navbar naar een icoontje met een drop-down, zodat de website ook 
bruikbaar blijft bij deze actie. 

Op het 'projects' gedeelte van de website heb ik er ook voor gekozen een react bootstrap component te integreren, 
dit omdat het renderen van alle afbeeldingen (die redelijk groot zijn) er voor zorgt dat de pagina wat traag zou kunnen zijn.
In de components op de projects-pagina zit er dan nog een carousel component. Deze werken allemaal, maar zou zoals gezegd dus wat traag 
kunnen zijn.

Op verschillende pagina's heb ik ook een gridsysteem gebruikt om de layout te verzorgen. 

Routing en het gebruik van de header en de footer over de globale website zijn duidelijk uit de code.

Er zijn ook sociale media links toegevoegd aan de website.
